# 🍔 Online Food Delivery System

## 📌 Project Overview
This project is a Java Object-Oriented Programming (OOP) application that simulates an online food delivery system.

The system manages different types of customer orders, calculates bills, applies delivery charges, and generates total platform sales using core OOP concepts.

## ✨ Features
- Manage Regular and Express Orders
- Calculate order bills automatically
- Apply standard delivery charges for regular orders
- Support express delivery with additional express fees
- Display customer and restaurant details
- Calculate total platform sales

## 🛠 Technologies Used
- Java
- Object-Oriented Programming (OOP)

## 💡 OOP Concepts Used
- Abstraction
- Inheritance
- Polymorphism
- Encapsulation
- Method Overriding
- Arrays of Objects

## 📷 Sample Output
The program displays:
- Customer Details
- Restaurant Details
- Menu Items
- Delivery Type
- Order Bill
- Total Platform Sales
